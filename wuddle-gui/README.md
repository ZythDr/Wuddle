# Tauri + Vanilla

This template should help get you started developing with Tauri in vanilla HTML, CSS and Javascript.

## App Icon

1. Replace `src-tauri/icons/app-icon.png` with your icon (square PNG with transparency, ideally 1024x1024).
2. Regenerate all platform icon files:

```bash
npm run icon
```

This updates titlebar/taskbar/bundle icons for Linux, Windows, and macOS builds.
